Research Interest

My principal research interest includes phylogenetics, population genetics, conservation genetics, molecular ecology.

Skills

Familiar with software used for the analysis of DNA sequences (Geneious, Sequencher, GeneStudio, DnaSp, Mega, MrModeltest, PAUP, MrBayes, PopArt, TCS).

Familiar with software used for the analysis of nuclear DNA and population genetics (GeneMarker, Microchecker, Cervus, ML-Relate, Structure, Structure Harvester, Arlequin, Poptree2, NeEstimator).

Familiar with SQL Database design and with relational tables in PostgreSQL.

Familiar with python scripting as well as with packages for data analysis and visualization (NumPy, Pandas, Matplotlib, Seaborn) in Jupyter Notebook.

Familiar with Linux command line.

Familiar with R packages for phylogenetic tree manipulation (ape, phytools, ggtree) and statistical analysis.

Familiar with R package Decipher for the design of qPCR primers.

Familiar with R packages for regression and classification analysis.
